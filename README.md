# Desafio Criativo — Prompt para Análise de Feedbacks de Clientes

Este repositório contém o prompt final desenvolvido para o desafio da DIO, cujo objetivo é usar IA para analisar feedbacks de clientes bancários e transformá-los em insights acionáveis para equipes de produto e atendimento.

## 📄 Conteúdo

- [`prompt-final-analise-feedbacks.md`](./prompt-final-analise-feedbacks.md) — Prompt final completo, incluindo:
  - Papel, objetivo e contexto da análise
  - Dados disponíveis e critérios de classificação
  - Formato de resposta esperado
  - Restrições de uso (dados, privacidade, honestidade da IA)
  - Demonstração prática com uma base de dados fictícia e o resultado gerado pela IA

## 🎯 Sobre o desafio

O desafio propôs construir um prompt em três etapas:

1. **Definir a intenção** — que tipo de feedback será analisado, para quem e com qual finalidade.
2. **Adicionar contexto e restrições** — dados disponíveis, critérios de análise e cuidados a serem tomados.
3. **Unir tudo em um prompt final** — testado com uma base de dados de exemplo para validar seu funcionamento.

## 🧪 Caso de uso

O prompt foi construído para um cenário bancário fictício, cobrindo feedbacks sobre aplicativo, Pix, cartão de crédito e atendimento via chat, com o objetivo de apoiar decisões de priorização no backlog de produto e atendimento.

## 🛠️ Como usar

Copie o conteúdo do prompt final e cole em qualquer ferramenta de IA generativa (ChatGPT, Claude, Gemini, etc.), fornecendo a base de feedbacks a ser analisada no formato descrito no prompt (ex: JSON com data, canal, texto do feedback, produto citado e nota de satisfação).
